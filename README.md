# ARTSOME — návrh nového webu

Kompletní responzivní prototyp webu pro malířství / natěračství ARTSOME s.r.o.

## Jak si web prohlédnout
Otevřete **`index.html`** v prohlížeči. Stránky jsou propojené, takže můžete proklikat celý web (Domů → Služby → Reference → Kontakt). Funguje i na mobilu (zmenšete okno prohlížeče nebo otevřete na telefonu).

## Struktura
- **index.html** — hlavní stránka (hero, služby, ukázka referencí, proměna před/po, recenze, poptávkový formulář)
- **reference.html** — plná fotogalerie s filtrováním podle typu práce + lightbox + sekce před/po
- **kontakt.html** — poptávkový formulář, kontaktní údaje, otevírací doba, mapa, časté dotazy
- **style.css** — sdílený design (barvy, typografie, komponenty)
- **assets/** — fotky

## Důležité k fotkám
Fotky v návrhu jsou **vyříznuté z původního webu** (reálné realizace ARTSOME) a slouží jako ukázka rozložení. Pro ostrou verzi doporučuji dodat **fotky v plném rozlišení** přímo z mobilu/foťáku — návrh je na kvalitní fotky připravený.

Místa, kde reálná fotka zatím chybí, jsou označena **tmavě šedým polem s popiskem** (např. „Plovoucí podlaha", „Nátěr plotu"). Sem se doplní reálné fotky, až budou k dispozici.

## Co bude potřeba doplnit při ostrém nasazení
- Reálné fotky ve vysokém rozlišení (hlavně reference a hero)
- Skutečné recenze (vytáhnout z 17 hodnocení na Firmy.cz)
- Funkční odeslání formuláře (napojení na e-mail)
- Vložení mapy (Google Mapy / Mapy.cz)
- Loga certifikátů (Firmy.cz, NejŘemeslníci, Poptávej.cz) v originále
